# 📚 Programação Orientada a Objetos em Python 2

Repositório destinado ao armazenamento de projetos, exercícios e atividades desenvolvidas na disciplina **Programação Orientada a Objetos em Python 2**, do curso de **Sistemas de Informação - UFSC**.

## 🎯 Objetivo

Consolidar e aplicar conceitos avançados de Programação Orientada a Objetos (POO) utilizando Python, incluindo:

- Encapsulamento
- Herança
- Polimorfismo
- Abstração
- Boas práticas de organização de código
- Estruturação de projetos

## 📂 Estrutura do Repositório

O repositório está organizado da seguinte forma:

/
├── aa1/
    ├── arquivos do aa1/
├── lista_1/
    ├── exercicio_01.py
    ├── ...
└── README.md


- Cada pasta representa uma lista de atividades ou projeto específico da disciplina
- Os códigos são independentes entre si
- Cada diretório pode conter seu próprio README com detalhes adicionais

## 🚀 Tecnologias Utilizadas

- Python 3.x
